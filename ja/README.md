## この本について

この本は、人間行動の基本原理を探求し、私たちの行動と決定に影響を与える心理的、社会的、生物学的要因を検討します。

### 対象読者

- 心理学と行動科学に興味を持つ学生
- 人間の動機付けを理解したい専門家
- 人間行動の科学について興味のある一般の方
