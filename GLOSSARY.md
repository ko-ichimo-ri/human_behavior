# 用語集 / Glossary

人間行動の分類プロジェクト全体（第1章：基礎編、第2章：応用編、第3章：AI・ロボット時代の生き方）の用語について、英語と日本語の対応をまとめた用語集です。

A Glossary of terms from the Human Behavior Classification Project (Chapter 1: Fundamentals, Chapter 2: Advanced Topics, Chapter 3: How to Live in the Age of AI and Robots) with English-Japanese correspondence.

## 第1章の基本的な構造的用語 / Chapter 1: Basic Structural Terms

### 人間活動の三要素 / Three Elements of Human Activity

| 日本語 | English | 備考 |
|--------|---------|------|
| 処理 | Processing | 主体が意図的に行う試行錯誤の過程 |
| 入出力 | Input/Output | 処理中に変わらないもの |
| 主体 | Agent | "Subject"も考えられるが、処理を主動的に行う存在という意味で"Agent"を採用 |
| 試行錯誤 | Trial and error | 処理の本質を表現 |

### 変更可能体：3つの変更可能性 / Modifiable Entities: Three Types of Changeability

| 日本語 | English | 備考 |
|--------|---------|------|
| 客体 | Object | "Objective"は形容詞で、名詞としては"Object"が適切。主体が変更できない客観的存在 |
| 個体 | Individual | 主体が1つ変更可能な対象。個人の内面や選択を表す |
| 社会 | Society | 主体が2つ以上変更可能な対象。複数の主体の相互作用を表す |
| 変更可能体 | Changeable entity | "Modifiable entity"も使用可能だが、変動性を重視して"Changeable"を採用 |

### 軸と成果：妥当性と発展性 / Axis and Results: Validity and Developability

| 日本語 | English | 備考 |
|--------|---------|------|
| 妥当性 | Validity | 判断や結論が論理的に正しいこと |
| 発展性 | Developability | 将来的に改善や応用が可能な性質 |
| 短期的な軸 | Short-term axis | 限定された時間スケールでの評価基準 |
| 長期的な軸 | Long-term axis | 長い時間スケールでの評価基準 |

### 判断軸：正否と真善美 / Judgment Axes: Correctness/Incorrectness and Truth/Goodness/Beauty

| 日本語 | English | 備考 |
|--------|---------|------|
| 正 | Correct | 判断や命題が正確で適切である |
| 否 | Incorrect | 判断や命題が誤りまたは不適切である |
| 真 | True | 命題の内容が現実と一致する |
| 偽 | False | 命題の内容が現実と一致しない |
| 美 | Beautiful | 視覚的または倫理的に優れて見える |
| 醜 | Ugly | 視覚的または倫理的に劣って見える |
| 善 | Good | 道徳的に正しく望ましい |
| 悪 | Evil | 道徳的に邪悪で望ましくない |

### 自由と束縛：3つの性質 / Freedom and Constraints: Three Properties

| 日本語 | English | 備考 |
|--------|---------|------|
| 主体性 | Agency | "Subjectivity"は主観性で異なる意味。主体が変更・選択できる自由さを表現するため"Agency"を採用 |
| 普遍性 | Universality | 主体の変更が困難で普遍的である性質 |
| 過重性 | Weight | "Dominance"よりも負担の大きさを表現する"Weight"を採用。一人では実現できない重さ |
| 自由 | Freedom | 束縛からの解放 |
| 束縛 | Constraint | 活動を制限する要因 |
| 双対性 | Duality | 互いに相補的な関係 |

### 感覚と認知 / Senses and Cognition

| 日本語 | English | 備考 |
|--------|---------|------|
| 感性 | Sensibility | "Sensitivity"は感度で異なる意味。人間の感受性と美的感覚を表現するため"Sensibility"を採用 |
| 知性 | Intellect | "Intelligence"は一般的な知能だが、思考と理解の能力を指すため"Intellect"を採用 |
| 理性 | Reason | "Rationality"は理性的思考ですが、より広く論理的推論と判断を表現するため"Reason"を採用 |
| 聴覚 | Hearing | "Auditory sense"より簡潔。音声知覚の能力 |
| 視覚 | Vision | "Visual sense"より簡潔。画像知覚の能力 |
| 触覚 | Touch | "Tactile sense"より簡潔。接触知覚の能力 |
| 嗅覚 | Smell | "Olfactory sense"より簡潔。臭気知覚の能力 |
| 味覚 | Taste | 化学的味覚知覚の能力。一般的で明確な用語 |
